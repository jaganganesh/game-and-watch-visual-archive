# Tricotronic

A visual archive of every **Nintendo Game & Watch** handheld electronic game released by **Nintendo** between **1980 and 1991**.

This repository documents the **complete Game & Watch catalog**, including classic LCD handheld games such as:

- Ball
- Fire
- Donkey Kong
- Chef
- Flagman
- Bomb Sweeper
- Green House
- Helmet

The goal of this project is to create a **single reference library for the Game & Watch series**.

# About Game & Watch

The **Game & Watch** series is a line of handheld electronic games created by Nintendo engineer **Gunpei Yokoi**.

Each Game & Watch device contains:

- A single LCD video game
- Digital clock functionality
- An alarm feature

These handheld systems were among the **first successful portable gaming devices**.

Many design ideas from Game & Watch later influenced Nintendo hardware such as:

- Nintendo Game Boy
- Nintendo DS

# Game & Watch ROMs

Game & Watch ROM files are available for archival and emulation purposes.

You can find a collection hosted on the Internet Archive:

https://archive.org/details/game-watch-roms

These ROMs can be used with emulators that support **Game & Watch systems**.

> This repository does **not host ROM files directly**.  
> It only provides documentation and visual references for the Game & Watch catalog.

# Game & Watch Hardware Series

Nintendo released several hardware designs during the Game & Watch era.

## Gold Series

An early Game & Watch hardware design with improved display brightness compared to earlier devices.

---

## Wide Screen Series

Devices with a **larger LCD screen** for improved gameplay visibility.

Popular titles include:

- Fire
- Chef
- Helmet

---

## Multi Screen Series

A two-screen design that later inspired the **Nintendo DS** handheld console.

Popular titles include:

- Donkey Kong
- Oil Panic

---

## Tabletop Series

Larger tabletop versions of Game & Watch games featuring color displays and arcade-style cabinets.

---

## Panorama Series

Games that used mirrors and angled screens to create a pseudo-3D visual effect.

# About Game & Watch

The **Game & Watch** line was Nintendo's pioneering series of handheld LCD electronic games, conceived by **[Gunpei Yokoi](https://en.wikipedia.org/wiki/Gunpei_Yokoi)** and released between **1980 and 1991**.

Game & Watch systems typically paired:

- A single LCD game
- A digital clock
- Alarm functionality in many models

The series helped define portable gaming and remains a foundational part of Nintendo's handheld history.

# What This Repository Includes

This project currently focuses on **visual reference material**, not software, ROM distribution, or emulation tooling.

Included in this repository:

- 40 JPEG images of Game & Watch titles stored under [`images/`](./images)
- A GitHub-friendly image gallery for quick browsing
- Documentation for contributing, supporting, and expanding the archive
- GitHub Sponsors funding configuration through [`.github/FUNDING.yml`](./.github/FUNDING.yml)

This repository does **not** include:

- ROM files
- Firmware dumps
- Emulators

The goal is to keep the project useful as a **Game & Watch image archive**, **collector reference**, and **retro gaming preservation resource**.

# How to Use This Repository

There are no build steps or runtime dependencies. You can browse the archive directly on GitHub or clone it locally for easier access to the image files.

```bash
git clone https://github.com/jaganganesh/game-and-watch-visual-archive.git
cd game-and-watch-visual-archive
```

Common use cases include:

- Quick visual reference for Game & Watch research
- Collection tracking and comparison
- Retro gaming articles, videos, and social posts
- Preservation and documentation projects
- Inspiration for future metadata, catalog, or restoration work

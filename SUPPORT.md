# Support Game & Watch Visual Archive

If this repository helps you, there are several simple ways to support it.

# Star the Repository

Starring the repository helps more people discover it through GitHub search, recommendations, and topic browsing.

# Share the Project

You can help by sharing the repository with:

- Retro gaming communities
- Nintendo history and preservation communities
- Game & Watch collectors
- Writers, video creators, and researchers
- Developers building retro gaming references or catalog projects

# Open Useful Issues

Helpful issues include:

- Missing Game & Watch titles
- Incorrect title names or labels
- Broken image links
- Better image replacements
- Documentation gaps
- Ideas for archive organization or metadata

# Contribute Improvements

Contributions are welcome.

Please remember:

- Open pull requests against `develop`
- Do not open pull requests directly against `main`

Contribution details:

- [CONTRIBUTING.md](./CONTRIBUTING.md)

# Sponsor the Work

If you want to support the project financially, you can sponsor the work through GitHub Sponsors:

- https://github.com/sponsors/jaganganesh

The repository already includes GitHub funding configuration, so sponsorship links can appear directly on GitHub.

# Other Helpful Support

You can also support the project by:

- Watching the repository for updates
- Forking the project
- Referencing it in articles, videos, checklists, or collection guides
- Linking to it from retro gaming resource pages
- Suggesting improvements that make the archive easier to discover

Every star, share, issue, contribution, and sponsorship helps the archive grow.
